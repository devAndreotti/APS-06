# 📘 Resumo do Trabalho APS

O trabalho de **Atividades Práticas Supervisionadas (APS)** tem como tema principal:
**"Aplicação de Visão Computacional no Monitoramento de Atividades Humanas: Estudo de Caso com Polichinelos".**

O objetivo geral é desenvolver um sistema computacional capaz de **reconhecer e contabilizar automaticamente polichinelos** realizados por uma ou mais pessoas, utilizando técnicas de visão computacional através da biblioteca **MediaPipe** (Google AI).

---

## 🎯 Objetivos Específicos

* [ ] Implementar um sistema de detecção de poses humanas com MediaPipe — **Responsável:**
* [ ] Desenvolver algoritmos para identificação e contagem automática dos movimentos — **Responsável:**
* [ ] Criar interface para processamento de vídeos gravados e captura em tempo real (webcam) — **Responsável:**
* [ ] Avaliar a precisão e eficiência do sistema — **Responsável:**
* [ ] Analisar limitações e propor melhorias para a solução — **Responsável:**

---

## ⚙️ Requisitos Técnicos

* **Linguagem:** Python 3.8 ou superior
* **Biblioteca principal:** MediaPipe (Google AI)
* **Bibliotecas auxiliares:** OpenCV, NumPy e Matplotlib
* **Funcionalidades obrigatórias:**
  * [ ] Detecção de poses em tempo real via webcam — **Responsável:**
  * [ ] Processamento de vídeos pré-gravados — **Responsável:**
  * [ ] Contagem automática de polichinelos — **Responsável:**
  * [ ] Exibição visual da contagem na interface — **Responsável:**
  * [ ] Suporte para detecção de múltiplas pessoas simultaneamente — **Responsável:**
